# KI-Literacy: Infrastruktur & Lokales Deployment

Begleit-Repository zum Modul **KI-Literacy**. 

An diesem Tag verlassen wir die reine Theorie und bauen unsere eigene, lokale KI-Infrastruktur auf. Wir lernen, wie man LLMs unabhängig von Cloud-Anbietern kontrolliert und erweitert.

## 🎯 Fokus von Tag 03: Deployment
- **Docker-Grundlagen:** Installation und Beherrschung der Container-Technologie.
- **Self-Hosting:** Bereitstellung von **OpenWebUI** als lokales Frontend.
- **Code Interpreter:** Erweiterung der KI um die Fähigkeit, Python-Code in einer gesicherten Umgebung (Jupyter) auszuführen.
- **Security:** Sicherung von lokalen Schnittstellen mittels Token-Authentifizierung.

## 📂 Dateien für heute
- `Befehle_Tag03.md`: Schritt-für-Schritt Anleitung zur Installation von Docker, OpenWebUI und dem Jupyter-Interpreter.
- `requirements.txt`: Die benötigten Python-Bibliotheken für den Code-Interpreter.
- `Linkliste.md`: Ressourcen zu Docker-Images und OpenWebUI Dokumentation.
