# Cheatsheet: Kamera- & Objektsteuerung im Film-Kontext

## 1. Kamerabewegungen (Camera Motion)
*Diese Begriffe steuern die Position und den Blickwinkel des "Beobachters".*

| Begriff | Beschreibung | Emotionale Wirkung |
| :--- | :--- | :--- |
| **Static / Fixed** | Kamera auf Stativ, völlig unbewegt. | Ruhe, Beobachtung, Fokus. |
| **Pan (Left/Right)** | Horizontale Drehung um die eigene Achse. | Überblick, Verfolgung von horizontaler Action. |
| **Tilt (Up/Down)** | Vertikale Drehung um die eigene Achse. | Erhabenheit (Tilt up) vs. Unterwerfung (Tilt down). |
| **Dolly In / Out** | Physische Fahrt der Kamera auf das Motiv zu/weg. | Zunehmende Intimität oder Distanzierung. |
| **Tracking / Trucking** | Physische Fahrt parallel zum Motiv (seitlich). | Dynamik, Begleitung einer Bewegung. |
| **Pedestal Up/Down** | Die gesamte Kamera bewegt sich vertikal (hoch/runter). | Perspektivwechsel ohne Winkeländerung. |
| **Zoom In / Out** | Optische Brennweitenänderung (keine physische Fahrt). | Aufmerksamkeit erzwingen, Raum verdichten. |
| **Dolly Zoom (Vertigo)** | Dolly In + Zoom Out (oder umgekehrt). | Schwindel, Schock, psychologische Irritation. |
| **Orbit / Arc** | Kreis- oder Halbbogenfahrt um das Motiv. | Hero-Shot, 3D-Exploration des Raums. |
| **Crane / Jib** | Kamera hebt/senkt sich im weiten Bogen von oben. | Epische Übersicht, "Gott-Perspektive". |
| **Handheld / Gimbal** | Gewollte Instabilität (Zittern/Wackeln). | Authentizität, Hektik, Dokumentar-Stil. |
| **Dutch Angle** | Kamera wird um die Längsachse gekippt. | Unbehagen, Wahnsinn, Desorientierung. |

## 2. Objektbewegung (Subject Motion)
*Aktionen innerhalb des Bildes bei (meist) statischer Kamera.*

| Bewegung | Fokus | KI-Challenge |
| :--- | :--- | :--- |
| **Gezielte Geste** | "waving", "pointing" | Fingerkonsistenz (Anatomie-Erhalt). |
| **Mimik** | "smiling", "crying" | Erhalt der Gesichtszüge (Identity). |
| **Micro-Motion** | "wind in hair", "breathing" | Flimmern im Hintergrund (Stability). |
| **Kinematik** | "running", "jumping" | Morphing-Risiko bei schnellen Posenwechseln. |

## 3. Zeit- & Technik-Parameter

*   **FPS (Frames Per Second):** 24 (Film), 30 (Web), 60+ (Slow-Mo/Sport).
*   **Motion Bucket / Strength:** Steuert die Intensität der generierten Bewegung.
*   **Temporal Consistency:** Die Fähigkeit der KI, Details über die Zeit stabil zu halten.
*   **Latent Drift:** Das schleichende "Vergessen" von Details über die Dauer des Clips.
