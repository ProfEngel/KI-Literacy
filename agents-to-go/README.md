# Modul: Agents To-Go

In diesem Modul geht es um die Portabilität von KI. Wir untersuchen, wie lokale Sprachmodelle und Agenten-Systeme auf mobilen Geräten oder als portable Instanzen genutzt werden können, um auch unterwegs volle Souveränität über die eigenen Daten zu behalten.

## Themen
*   **Hermes**: Die mobile Schnittstelle für lokale Intelligenz.
*   **OpenClaw**: Einblicke in portable Agenten-Lösungen.
*   **On-Device Processing**: Chancen und Grenzen aktueller mobiler Hardware.
