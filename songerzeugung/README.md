# Modul: Songerzeugung mit KI

Dieses Modul widmet sich der systematischen Erstellung von Musik und Gesang mittels generativer KI. Wir untersuchen die Transformation von Text zu Audio (Text-to-Audio) und die Kontrolle über Genre, Stimmung und instrumentale Komplexität.

## 🎯 Lernziele
*   Verständnis der Funktionsweise von Audio-Diffusionsmodellen.
*   Beherrschung von Prompting-Techniken für komplexe Song-Strukturen (Strophe, Refrain, Bridge).
*   Integration von Lyrik-Generierung und Komposition.
*   Nutzung von Multimodalen Workflows (z.B. Song zu Video).
