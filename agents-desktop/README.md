# Modul: Agents Desktop

Dieses Modul fokussiert sich auf die Integration von KI-Agenten in die professionelle Desktop-Arbeitsumgebung. Wir untersuchen, wie **Agentic Coding** den Softwareentwicklungsprozess verändert und wie Tools wie VS Code und Antigravity als Co-Piloten fungieren.

## Themen
*   **VS Code & Codex**: Die IDE als Host für KI-Intelligenz.
*   **Antigravity Workflows**: Systematisches Lösen komplexer Coding-Aufgaben durch Agenten.
*   **Local LLMs for Coding**: Performance und Datenschutz auf dem eigenen Rechner.
